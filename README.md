# astart-cli

> A beautiful javascript scaffold for seed application

## ✨ Feature

- One common scaffold cli to init seed application with best solutions

## 🚀 Start

### install

```bash
npm i -D astart-cli # OR yarn add -D astart-cli
```

```js
  "scripts": {
    "astart": "astart"
  },
```

```bash
npm run astart # OR yarn astart
```
